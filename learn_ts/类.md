# 类

```typescript
// 公共成员修饰符public  在类外和类内都可以访问
// 私有成员修饰符private 只可以在当前类的内部使用
// 受保护成员修饰符protected 不能在类的外部使用，但是可以在派生类中使用

// 类的类型定义可以参考接口定义类
```

注意：构造函数也可以被标记成 `protected` 。 这意味着这个类不能在包含它的类外被实例化，但是能被继承
