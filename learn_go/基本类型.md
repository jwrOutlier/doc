# 基本类型

#### 布尔类型

```go
var bool = false
```

#### 整型

```go
var num int = 10
var num2 int8 = 10
fmt.Println(num)
fmt.Println(num2)
```

#### 浮点数

```go
var f_num float32 = 10.88888888
fmt.Println(f_num)
```

#### 字符类型

```go
var chars byte = 'a'   // 表达ANSCII字符
var un rune = '\u4e2d' // 可以表达Unicode字
var str string = "hello"
fmt.Println(chars)
fmt.Println(un)
fmt.Println(str)
```
